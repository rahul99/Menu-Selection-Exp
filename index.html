Pre Experiment Questionnaire

Introduction

You have been chosen as part of an experiment to analyze two types of menus. You will be asked to select a set of objects from various
categories using the two types of menus.

For this experiment you will also be asked to use a mouse and the trackpad on your laptop.
Please attach your USB mouse to the laptop now and keep it attached till the end of the experiment for convenience.

The experiment will take about 5-10 minutes. Please complete the experiment in one sitting.

The interface you will be working with is shown below

Trial will show the current trial number and the total number of trials to be done.

Input device will show the input device you need to use for the current trial.
It will show either Mouse or Trackpad. Please use the correct device accordingly.

Menu Type will show the type of menu that you will use in the current trial.
Your task is to select the item shown under Please Select.

Menu Depth is the depth at which the item you must select resides.

These fields will be shown at the top of the page for each trial. Please look at these carefully before selecting the item in each trial.

Usage of menus:

Radial menu:

•	Right click once to popup the menu
•	Left click on an item to select it
•	Right click to reset menu

Marking menu:

•	Hold left mouse button to popup menu
•	Drag pointer to category and release left mouse button to select it
•	Release left mouse button to close menu
•	Advanced users can draw the entire stroke to select item instead of waiting for the menu popup.
If at any time you need help on how to use the menus, click on the help button available at the top of the page for each trial.

NOTE: The first 4 trials are for practise to familiarize you to the interface and the menus. They will not be counted towards the final results.
